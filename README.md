# Plentifully Whitepaper (v1.2)

This repository hosts the Plentifully Whitepaper for public viewing via GitHub Pages.

## Files
- `index.html` – Full-screen embedded PDF viewer
- `plentifully-whitepaper.pdf` – The whitepaper PDF
- `.nojekyll` – Ensures GitHub Pages serves static assets without Jekyll processing

GitHub Pages will publish at:
```
https://plentifully.github.io/whitepaper/
```
- Direct PDF link:
```
https://plentifully.github.io/whitepaper/plentifully-whitepaper.pdf
```
- Embedded viewer:
```
https://plentifully.github.io/whitepaper/
```

> If your org or repo name differs, adjust the URLs accordingly:
> `https://<org-or-user>.github.io/<repo-name>/plentifully-whitepaper.pdf`

## Optional
- Add a custom domain in **Settings → Pages** (creates a `CNAME` file in the repo).
- If you prefer serving only the raw PDF, you can omit `index.html` and link directly to the PDF path above.
